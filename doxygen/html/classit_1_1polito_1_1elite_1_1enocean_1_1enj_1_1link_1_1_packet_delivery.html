Detailed Description

This class provides a utility for sampling a given queue containing ESP3Packets encapsulated into PacketQueueItems, and to deliver such packets to registered PacketListeners. It allows to decouple reception timing from event handling times, thus acting as "buffer" layer between low level events and high level packet handling.
