Detailed Description

A class representing the D2-01 family of EnOcean Equipment Profiles. Cannot be directly istantiated, instead specific classes matching the real device EEP must be used.

D2.01 CMD 0x02, Implements the CMD 0x02 of the D2.01 EnOcean Equipment Profile. It allows to configure one or all channels of a single actuator. No response timing is defined / required.

Recommendation: In case the device implements an inner order for dim timers, this order should be from "dim timer1"(fast) to "dim timer3"(slow). the configured time shall always be interpreted for a full range (0 to 100%) dimming.

Parameters
connectionThe {} link to be used for sending the command, packet encapsulation will be performed at such level.
deviceAddressThe low level address of the device to which the configuration command must be sent.
channelIdThe id of the channel to be configured.
localControlEnables / disables the local control.
overCurrentShutDownConfigures the behavior of the device in case of over-current shutdown.
resetOverCurrentShutDownConfigures the device behavior when resetting after an over-current shutdown.
userInterfaceIndicationSets the user interface mode, from day to night.
powerFailureSets the device behavior upon a power failure event.
defaultStateSets the default device state, to be restored when the device powers up.
dimTime1The dim timer 1.
dimTime2The dim timer 2.
dimTime3The dim timer 3.

D2.01 CMD 0x05, Implements the CMD 0x05 of the D2.01 EnOcean Equipment Profile. It configures the energy and power measurement of one or all channels of an actuator.

Parameters
connectionThe object which enables command transmission over the physical network.
deviceAddressThe address of the destination device.
reportMeasurement0b0 query only, 0b1 auto reporting and query.
resetMeasurement0b0 not active, 0b1 trigger signal.
measurementMode0b0 energy measurement, 0b1 power measurement.
channelIdOutput channel from 0x00 to 0x1D.
measurementDeltaLSBMeasurement delta to be reported (LSB) 0-4095.
unitUnit of measure, 0x00 Energy (Ws), 0x01 Energy (Wh), 0x02 Energy (kWh), 0x03 Power (W), 0x04 Power (kW), 0x05..0x07 not used.
measurementDeltaMSBMeasurement delta to be reported (MSB) 0-4095.
matMaximum time between to subsequent actuator messages (10-2550s).
mitMinimum time between to subsequent actuator messages (0-255s).

D2.01 CMD 0x1 Implements the CMD 0x1 of the D2.01 EnOcean Equipment Profile, which allows setting the output level of a D2.01 device (On, Off and Dimming are supported).

Parameters
connectionThe {} link to be used for sending the command, packet encapsulation will be performed at such level.
dimValueThe required dimming value (low-level, in byte).
ioChannelThe output channel to be controlled (each device may have more than one channel, each controllable independently from the others).
outputValueThe required ouput value. A byte representing a percentage between 0 (0x00) and 100 (0x64).

D2.01 CMD 0x03, Implements the CMD 0x03 of the D2.01 EnOcean Equipment Profile. It requests the status of one or all channels of an actuator.

Response timing: An actuator status response message shall be received within a maximum of 300ms from the time of transmission of this message. In case no such response is received within this time frame the action shall be treated as completed without results.
