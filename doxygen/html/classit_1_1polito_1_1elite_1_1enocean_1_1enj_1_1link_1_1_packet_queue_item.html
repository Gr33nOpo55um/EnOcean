Detailed Description

A class modeling a queue item containing an EnOcean ESP3Packet, and a (re)transmission counter. It is mainly used in transmission queues to account for possible re-transmissions. Defines a default retransmission counter, which can be redefined programmatically, or overridden by using the proper constructor.
